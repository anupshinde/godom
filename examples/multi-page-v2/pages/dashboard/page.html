This dashboard shows four independent islands running in one Go process — a counter, an analog clock, a simulated system monitor, and a 3D solar system. Counter, clock, and monitor share a single *counter.State pointer, so incrementing the counter auto-refreshes the other two views. The current time right now is — rendered by a minimal island using inline Island.TemplateHTML with no filesystem at all.
