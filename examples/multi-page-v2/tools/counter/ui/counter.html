What this demo shows

  • Per-island AssetsFS — the counter package embeds its own filesystem via //go:embed ui, so the Go code and HTML ship together as a portable unit.
  • Local sibling partials — <decrement-button/> and <increment-button/> resolve by filename (decrement-button.html, increment-button.html) from the island's own FS. No registration needed — filesystem lookup.
  • Shared partial — this wrapper is <info-note>, registered once via eng.UsePartials(partialsFS, "partials") in main.go. Available to every island without per-island setup.
  • Children via <g-slot/> — the text you're reading right now was passed as children of <info-note>; the partial has a <g-slot/> marker where it gets inserted.
