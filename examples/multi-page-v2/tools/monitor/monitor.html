What this demo shows

  • Per-island AssetsFS — monitor embeds its single HTML file (//go:embed monitor.html) alongside the Go code.
  • Plugin system — Chart.js is wired in via godom's plugin mechanism. eng.Use(chartjs.Plugin) registers a named JS bridge; g-plugin:chartjs="CPUChart" hands Go-side data to that bridge on mount and on each patch.
  • Shared pointer state — same *counter.State as clock; the Count/Step at the bottom reflects the counter tile automatically.
  • <info-note> — shared partial, slotted content — same pattern as counter and clock.
